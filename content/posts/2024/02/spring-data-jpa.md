JPA（Jakarta Persistence API）是一种基于 ORM（Object-Relational Mapping，对象关系映射）技术的 Java EE 规范，用于在 Java 应用程序和关系型数据库之间持久化、访问和管理数据。JPA 规范提供了一系列注解和 API 以将 Java 对象映射为数据库表、将表之间的关系映射为 Java 实体之间的关系、将查询与更新等数据库操作封装为普通的 Java 方法，从而大大简化了 Java 应用程序数据持久化层的开发。
